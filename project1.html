01. 이 프로젝트는 문제를 이렇게 해결했어요!

소상공인의 매출 데이터를 분석하여 생존 전략을 시각화하다

02. 프로젝트 제목

소상공인 생존 분석: 지역별 매출 데이터를 통한 인사이트 도출

03. 개요

이 프로젝트는 코로나 이후 소상공인의 매출 변화를 분석하고, 생존 전략을 데이터로 도출하기 위한 목적으로 수행되었습니다.

04. 사용 도구 및 기법

  • Python: 데이터 분석 및 시각화
  • Pandas, Seaborn, Matplotlib
  • Google Colab, Excel, Tableau
# 라이브러리 임포트
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

05. 사용 데이터

출처: 공공데이터포털
컬럼: 지역, 업종, 매출액, 날짜 등

# 데이터 불러오기
sales_df = pd.read_csv("sales_data.csv")
sales_df.head()

06. 분석 과정

수집 → 정제 → 분석 → 시각화 순으로 진행했습니다.

# 결측치 제거
sales_df.dropna(inplace=True)

# 지역별 매출 시각화
sns.barplot(data=sales_df, x='지역', y='매출')

07. 주요 결과

강남구는 매출 회복 속도가 빨랐으며, 소형 음식업종이 회복력이 높았다는 점을 확인했습니다.

# 월별 매출 추이 시각화
plt.plot(sales_df['월'], sales_df['매출'])
plt.title("월별 매출 변화")

08. 내가 맡은 역할

데이터 수집, 정제, 분석, 시각화 전 과정을 주도적으로 수행하였고 보고서 및 발표 자료를 직접 작성했습니다.

09. 느낀 점 / 한계

실제 데이터를 다루며 실무 경험을 쌓을 수 있었고, 분석의 신뢰성을 확보하는 것이 중요하다는 점을 배웠습니다. 시계열 분석 기법의 미숙함이 아쉬웠습니다.
